# Haotian Bian Academic Homepage

This repository hosts Haotian Bian's academic homepage at:

https://psyhtbian.github.io

The site is built with the Academic Pages template for GitHub Pages.

## Main Content

* `_pages/about.md`: homepage biography and research interests
* `_pages/research.md`: research overview
* `_pages/cv.md`: CV page
* `_publications/`: publication records
* `images/profile.jpg`: profile photo
* `_config.yml`: site-wide settings and profile links

## Local Preview

If Ruby and Bundler are available:

```bash
bundle install
bundle exec jekyll serve -H localhost -P 4000
```

Then open `http://localhost:4000`.
